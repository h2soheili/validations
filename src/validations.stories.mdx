## نحوه استفاده از متد های کاربردی

<p>
  نحوه استفاده ، به این شکل است که ولیدیشن موردنظر را import نموده و در فرم
  موردنظر، استفاده میشود.
</p>

  از توابع زیر برای ولیدیشن های پروژه می توان استفاده کرد. لازمه استفاده از این
  ولیدیشن ها ، استفاده از پکیج formik میباشد.{" "}
</p>
